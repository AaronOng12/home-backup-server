## Project Title
Home Backup Server with Raspberry Pi 4 and ESP32-C3

## Summary
The goal of this project is to build a small personal backup server for storing photos, short videos, and document files. The system is centered around a Vilros Raspberry Pi 4 kit, which will handle storage and file-sharing tasks, while an ESP32-C3 is included for future hardware expansion and control features.

## Why I Chose This Project
I wanted to create a practical embedded and computing project that is useful in everyday life while also helping me build experience with system design, Raspberry Pi setup, storage management, and hardware integration.

## Initial Scope
- Local file backup
- Simple and low-cost starting version
- Expandable design for future storage upgrades
- Clear documentation of progress and design decisions

## Future Plans
- Add SSD or larger storage
- Improve the enclosure and organization
- Explore ESP32-based control or status features
- Create a cleaner user experience for uploads
