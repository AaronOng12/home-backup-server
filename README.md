# Home Backup Server with Raspberry Pi 4 and ESP32-C3

## Overview
This project is a small personal backup server for storing iPhone photos, short videos, and document files from a Mac. It uses a Vilros Raspberry Pi 4 kit as the main server platform and an ESP32-C3 for future hardware control and expansion.

## Goals
- Build a simple local backup server
- Learn Raspberry Pi file sharing and storage setup
- Document project progress professionally
- Leave room for future SSD and custom hardware expansion

## Hardware
- Vilros Raspberry Pi 4 kit
- ESP32-C3 development board
- Jumper wires
- microSD card
- Future upgrade: SSD or external storage

## Planned Architecture
- Raspberry Pi handles storage and file server tasks
- ESP32-C3 handles future control and status features
- Mac and phone connect to the Pi for file upload
